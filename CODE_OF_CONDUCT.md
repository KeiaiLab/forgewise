# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## Reporting

Report violations to: `conduct@keiailab.org` (or `security@keiailab.org` for
issues involving credentials or sensitive data).

## Enforcement

See the Contributor Covenant for enforcement guidelines.

을 행동 강령으로 채택합니다. 위반 신고는 `conduct@keiailab.org` 로 보내주세요
(자격 증명 또는 민감 데이터 관련 사안은 `security@keiailab.org`).
